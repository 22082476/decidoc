# DeciDoc

DeciDoc is een command-line tool (CLI) voor het vastleggen en documenteren van
ontwerp-, technische en leerkeuzes tijdens een stage of project.  
De tool genereert en beheert een gestructureerd Markdown-document met een
overzichtstabel en per keuze een gedetailleerde toelichting.

DeciDoc is ontwikkeld om verantwoording en reflectie laagdrempelig vast te leggen
en later eenvoudig te kunnen hergebruiken in een eindverantwoordingsdocument.

---

## Functionaliteiten

- Initialiseren van een standaard keuzelog (`.md`)
- Automatisch genereren van keuze-ID’s
- Overzichtstabel met klikbare verwijzingen
- Vastleggen van context, overwegingen, keuze, motivatie en reflectie
- Configuratie van standaard documentpad
- Reproduceerbaar en platformonafhankelijk
